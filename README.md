## ###### Shortcut Install

```bash
pnpm add react-i18next i18next sass antd @reduxjs/toolkit react-redux react-router-dom
```

## Install i18next

```bash
pnpm add react-i18next i18next
```

## Install SASS

```bash
pnpm add sass
```

## Install Ant Design

```bash
pnpm add antd
```

## Install Redux ToolKit

```bash
pnpm add @reduxjs/toolkit react-redux # for Redux Toolkit:
```
